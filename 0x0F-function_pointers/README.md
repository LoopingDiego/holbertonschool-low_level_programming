# Function pointers

Fucntion pointers are used for dynamic function behaviour

## Topics
* What are function pointers and how to use them
* What does a function pointer exactly hold
* Where does a function pointer point to in the virtual memory

## Content
### 0. print name
prints a name using a function pointer

### 1. array iterator
 
executes a function given as a parameter on each element of an array.

### 2. int index

searches for an integer in an array and returns its index

### 3. main

performs simple operations like a calculator
